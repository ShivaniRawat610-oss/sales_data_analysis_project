# 📊 Sales Data Analysis Project

## 📌 Overview
This project focuses on analyzing sales data to uncover key business insights such as trends, profitability, and product performance using Python.



## 🎯 Objective
The objective of this project is to:
- Analyze sales trends over time  
- Identify top-performing products  
- Detect loss-making products  
- Understand regional and category performance  
- Provide actionable business insights  



##  Tools & Technologies
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  

## 📂 Dataset Features
The dataset includes:
- Order ID, Order Date, Ship Date  
- Customer Details  
- Product Details (Category, Sub-category, Product Name)  
- Sales, Profit, Quantity  
- Shipping Cost, Order Priority  
- Region, Market  

## 🧹 Data Cleaning
- Converted date columns into datetime format  
- Handled missing values  
- Converted numeric columns  
- Removed duplicates  

## ⚙️ Feature Engineering
Created new features:
- Month  
- Year  
- Shipping Days  
- Profit Margin  

## 📊 Key Performance Indicators (KPIs)
- Total Sales  
- Total Profit  
- Total Orders  
- Total Quantity  
- Average Order Value  
- Profit Margin  

## 📈 Analysis Performed
- Year-wise and Monthly sales trends  
- Region and Market analysis  
- Category and Sub-category performance  
- Pareto Analysis (Top products contribution)  
- Shipping cost vs Profit analysis  
- Customer segmentation  

## 🔍 Key Insights
- Sales show seasonal trends across months  
- Some products generate losses (negative profit margin)  
- Top products contribute majority of revenue (Pareto principle)  
- Shipping cost impacts profitability  
- Performance varies across regions and categories
  
## 💡 Recommendations
- Focus on high-performing products  
- Improve or remove loss-making products  
- Optimize shipping cost  
- Target high-value customers  
- Improve low-performing regions  

## 🏁 Conclusion
This project demonstrates how data analysis can help businesses make informed decisions, improve profitability, and optimize operations.
